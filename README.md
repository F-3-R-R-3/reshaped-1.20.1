ReShaped adds a radial menu that allows you to quickly and intuitively switch between different block shape variants, avoiding unnecessary inventory hassle.  
It also introduces a large collection of additional shape variants, giving builders more flexibility and control when designing structures.

![Features banner](https://cdn.modrinth.com/data/cached_images/2132a5a816f48ff0fab4cfa5c1e0e2c24d474b84.png)

| Quick-to-use radial menu                                                                                                                                                                                                                                                                                                  |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| The radial menu displays all available shape variants of the block you are currently holding in your main hand and lets you swap between them instantly.<br> <kbd>Default keybind: Left Alt</kbd>![Image of the radial menu](https://cdn.modrinth.com/data/cached_images/abbfbed18a4b8065ece13be26d92f7b782d009e9_0.webp) |

| Automatically generated shape variants for vanilla and modded blocks                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Includes vertical slabs, vertical stairs, steps, corner blocks, and more. Additional block types, such as panel blocks, are planned for the future. ![Some possible block combinations](https://cdn.modrinth.com/data/cached_images/8feefc95304d2ef2a2fffe259fdf22c011ac6641.png) Example with another mod installed (Create mod in this case): ![All blocks when the Create mod is installed](https://cdn.modrinth.com/data/cached_images/7ccbb67a0f9955fd4716b1807fcc48f9cb7ca58b.png) Quickly preview all available blocks in creative mode using: `/reshaped place_all` |

| Mixable blocks                                                                                                                                                                                               |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Almost all block shapes are mixable, allowing the use of multiple materials within a single block. ![Mixed blocks](https://cdn.modrinth.com/data/cached_images/b33aa78aa40521cda29b0e3c71d80cb59700b1c8.png) |
